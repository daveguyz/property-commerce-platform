Embed live property auctions, listings, and bidder accounts on your WordPress site.

== Description ==

Property Commerce Platform brings a full property auction system to your
WordPress site with three shortcodes and two Gutenberg blocks. The heavy
lifting — live bidding, deposits, KYC, purchase agreements — runs on the
Property Commerce Platform backend; your site just embeds the widgets,
which inherit your theme's fonts and colours.

= Shortcodes =
* [pcp_auction_listing] — auction grid with Live/Upcoming/Closed tabs
* [pcp_auction_room lot_id="..."] — live bidding room
* [pcp_account] — bidder account dashboard

== Installation ==

1. Upload the plugin and activate it.
2. Go to Settings → Property Commerce.
3. Enter your Platform API URL, Tenant ID, and API key from app.propertycommerce.io.
4. Click "Test connection".
5. Add [pcp_auction_listing] to any page.
